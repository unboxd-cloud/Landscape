# Unboxd Cloud Landscape

The Unboxd Cloud Landscape is a structured map of technologies, protocols, platforms, and standards used to build governed cloud-native agent systems.

It helps answer four questions:

1. What belongs in the stack?
2. Why does it belong?
3. Which layer does it serve?
4. What is the maturity and role of each component?

## Core layers

- Identity & Access
- Agent Runtime
- Policy & Governance
- Data & Memory
- Observability
- Protocols
- Commerce & Payments
- Developer Experience
- Edge & Sovereign Cloud
- Storage & Filesystems

## Design intent

This landscape is built for practical platform decisions. It is not a marketing directory. Every entry should have a clear architectural role.
